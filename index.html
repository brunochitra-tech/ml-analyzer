☁️ Sincronización en la nube — acceso desde cualquier PC

⚫ Firebase no configurado
Paso 1 — Crear la base de datos gratuita en Firebase:
  1. Entrá a console.firebase.google.com con tu cuenta de Google
  2. Clic en Agregar proyecto → poné cualquier nombre → Continuar × 2
  3. En el menú lateral: Build → Realtime Database → Crear base de datos → Modo de prueba → Habilitar
  4. En ⚙ Configuración del proyecto → General → Tu app → clic en </> (Web) → Registrar app (cualquier nombre) → te muestra el objeto firebaseConfig
  5. Copiá ese objeto JSON y pegalo abajo → clic Conectar
Paso 2 — Publicar el sitio para compartir el link:
Subí los 3 archivos (index.html, styles.css, app.js) a netlify.com → Deploy manually → arrastrá la carpeta. Obtenés un link gratuito tipo https://tuapp.netlify.app para compartir con quien quieras.
💡 La configuración queda guardada en este navegador. Una vez conectado, cualquier reporte o lista que cargues se sincroniza automáticamente para todos los usuarios que abran el link.

Ayuda rápida

1. Cargá la lista de precios una vez (queda guardada en localStorage).
2. Cargá un reporte XLSX de Mercado Libre — hoja Ventas AR, header en fila 6.
3. Revisá el Dashboard, Control de precios, Logística y Auditoría.
4. Exportá cualquier tabla a CSV con los botones ⬇ CSV.
5. Para comparar con otro período, simplemente cargá un nuevo reporte: reemplaza al anterior pero la lista de precios se mantiene.
